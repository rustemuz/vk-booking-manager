О боте

Бот «ХАУС» — это мощный инструмент для автоматизации бронирований уютных комнат для отдыха, встреч и мероприятий. Созданный в ярких жёлто-чёрных тонах, он сочетает в себе удобство, функциональность и современный подход к управлению бронированиями.

  • Создан за 100 часов напряжённой работы.
  • Содержит более 5000 строк кода.
  • Во время тестирования сгенерировал свыше 4000 логов действий.
  • Всего за это время было создано 8 версий бота.
  • Общий вес всего проекта с учетом фото: 8,73 МБ

Возможности бота

«ХАУС» предоставляет пользователям и администраторам удобный интерфейс для бронирования комнат и управления ими. Вот основные функции:

  • Бронирование комнат с выбором даты, времени, длительности и количества человек.
  • Проверка доступности слотов в реальном времени.
  • Гибкая система ценообразования: разные тарифы для будней и выходных, скидки за длительное бронирование.
  • Поддержка тематических комнат для различных мероприятий: от вечеринок до спокойных кинопросмотров.
  • Уведомления пользователей и администраторов о статусе брони (создание, подтверждение, отклонение).
  • Интеграция с базой данных SQLite для хранения информации о бронированиях, пользователях и админ-чатах.

Админ-панель

Для администраторов реализована мощная панель управления с широкими возможностями:

Основные функции админ-панели:

  • Просмотр списка ожидающих и активных броней.
  • Подтверждение или отклонение заявок на бронирование.
  • Удаление броней по ID.
  • Просмотр статистики: количество броней, выручка, средняя длительность, популярные комнаты.
  • Доступ к логам действий администраторов.
  • Экспорт данных в Excel с аналитикой и графиками (распределение броней, популярность комнат, активность по дням).
  • Автоматическая регистрация участников админ-чата как администраторов.
  • Мониторинг участников админ-чата с уведомлениями и удалением прав при выходе из чата.

Команды админ-панели:

  • /list — Показать ожидающие брони.
  • /list_bookings — Показать все активные брони.
  • /delete_booking — Удалить бронь по ID.
  • /stats — Показать статистику бронирований.
  • /logs — Показать лог действий администраторов.
  • /peerregister — Зарегистрировать чат как админ-чат.
  • /excel — Экспортировать данные в Excel.
  • /help — Показать справку по командам.

Технические детали

Бот построен на современных технологиях и оптимизирован для высокой производительности:

  • Использует библиотеку vkbottle для взаимодействия с VK API.
  • Хранит данные в базе SQLite с таблицами для пользователей, броней, админов, чатов и логов.
  • Поддерживает асинхронное программирование с asyncio.
  • Обеспечивает логирование через logging для отладки и мониторинга.
  • Генерирует Excel-отчёты с помощью pandas и xlsxwriter.
  • Автоматически проверяет участников админ-чата каждые 5 минут.
  • Реализована защита от дублирования броней и пересечений по времени.
